## Récuperer le code et executer les commandes

`git clone https://github.com/gabykant/miage-python-group5.git`

``cd miage-python-group5`

`python -m venv venv`

`cd venv/Scripts`

_Saisir activate et valider_
`activate`

`pip install requirements.txt`

## Lancer le serveur

`python main.py`

## L'application demarre sur le port 8000. Ouvrir son navigateur à l'adresse http://localhost:8000
